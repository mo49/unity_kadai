# 制作物の説明

## 課題1

### Shooting_Game > hill_stage

敵に数秒間視線を合わせると、破壊パーティクルの発生、スコアの更新。

破壊された後は、ランダムな位置にリポップ。

- マウス操作（カメラ移動）
 - 左：左へ移動
 - 右：右へ移動
 - 中：位置リセット
  
- キーボード操作（敵のアニメーション）
 - W：Walk
 - R：Run
 - J：Jump
 - A:Attack

### Museum > sao

ウォークスルー型の美術館。

美術館の骨格はBlenderで生成。
